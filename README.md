Описание проекта
ServiceTrack — это приложение на JavaFX, предназначенное для автоматизации процесса записи автомобилей на техническое обслуживание и ремонт в автосервисе.

Программа предназначена для сотрудников сервисного центра и позволяет:

1.Эффективно управлять записями: Добавлять, просматривать и изменять заявки на ремонт.

2.Систематизировать данные клиентов и автомобилей: Хранить информацию о владельцах и их транспортных средствах.

3.Указывать перечень работ и статус заказа: Отслеживать текущий этап ремонта (например, "Принят", "В работе", "Готов к выдаче").

4.Упростить взаимодействие с клиентами: Быстро находить историю обращений и данные по автомобилю.

Основные функции:

1.Внесение данных о клиенте (ФИО, контакты).

2.Регистрация данных об автомобиле (марка, модель, VIN, гос. номер).

3.Оформление заявки с описанием проблемы и списком необходимых работ.

4.Отслеживание статуса выполнения заявки.

5.Просмотр истории всех заявок.

Установка и запуск

Предварительные требования:

1.Убедитесь, что на вашем компьютере установлена Java JDK 17 или новее.

(Проверить можно в командной строке: java -version)

2.(Опционально) Рекомендуемая среда разработки: IntelliJ IDEA 

Пошаговая инструкция
Склонируйте репозиторий:

bash
(git clone https://github.com/Mikhai19/ServiceTrack.git )
(cd ServiceTrack)

Откройте проект в вашей IDE:

Файл -> Открыть -> Выберите папку ServiceTrack.

Запустите приложение:
 (через IDE): Найдите главный класс src/main/java/com/servicetrack/Main.java и запустите его.

1. Создание новой заявки
При запуске вы увидите главное окно с списком существующих заявок.

Нажмите кнопку "Новая заявка".

Заполните форму:

Вкладка "Клиент": ФИО, номер телефона.

Вкладка "Автомобиль": Марка, модель, VIN.

Вкладка "Работы": Опишите проблему и выберите необходимые услуги из списка.

Нажмите "Сохранить". Новая заявка появится в общем списке.


2. Изменение статуса заявки

В главном окне выберите заявку из списка.

Нажмите кнопку "Изменить статус".

В появившемся диалоговом окне выберите новый статус из выпадающего списка (например, "В работе").

Нажмите "Подтвердить". Статус заявки обновится.


Технические требования:

Язык программирования: Java 17+

Минимальная версия ОС: Windows 10, macOS 10.15, Linux (с поддержкой JavaFX)

JVM: Требуется установленная Java Runtime Environment (JRE) версии 17 или выше.
