# ServiceTrack
ServiceTrack-приложение на JavaFX для автосервисов. Позволяет управлять записями клиентов, отслеживать статусы ремонта и хранить историю заявок.  Основные функции:  Добавление клиентов и автомобилей ,Создание заявок на ремонт ,Отслеживание статусов ("Принят", "В работе", "Готов") ,Просмотр истории обращений
